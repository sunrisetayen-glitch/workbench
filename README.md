# 灵感工作台

**自媒体灵感集中管理 + 跨平台参考搜索**

一个可安装到手机主屏的 PWA 工作台，手动添加你在各平台收藏的帖子/视频，点击即可看到库内相似内容，以及一键跳转各平台搜索类似内容。

## 功能

- **添加收藏**：粘贴链接、选平台、写标题/标签/笔记
- **卡片网格**：按平台筛选、标签云筛选、关键词搜索
- **详情弹窗**：打开原帖 / 编辑 / 删除
- **本地相似推荐**：基于标签 + 标题分词（中文），自动找出库里相似内容
- **在线搜索推荐**：一键跳转 10 个平台搜索类似内容（小红书 / 抖音 / B站 / 微博 / 知乎 / YouTube / Twitter / 豆瓣 / Instagram / TikTok）
- **数据备份**：导出 / 导入 JSON（换机无缝迁移）
- **离线可用**：PWA 安装后断网也能浏览已缓存内容

## 支持平台

小红书 · 抖音 · B站 · 微博 · 知乎 · YouTube · Twitter/X · 豆瓣 · Instagram · TikTok

## 本地运行

```bash
# 进入项目目录
cd /workspace

# 启动静态服务器（任意端口）
python3 -m http.server 8000

# 浏览器打开
# http://localhost:8000
```

手机访问同一局域网的 `http://<电脑IP>:8000` 即可使用。

## 部署到公网

把 `/workspace` 下所有文件推到任意支持静态托管的平台：

| 平台 | 方式 |
|------|------|
| GitHub Pages | 推到仓库 → Settings → Pages → main 分支 |
| Netlify | 拖拽文件夹或连接 Git |
| Vercel | 导入项目 |
| Cloudflare Pages | 连接 Git 或上传 |

部署后在手机浏览器访问 → 选择「添加到主屏幕」→ 完成安装。

## 数据存储与备份

- 数据存在浏览器 IndexedDB 中，**完全本地**，不经过任何服务器
- 点击顶栏 ⬇ 导出 JSON 备份文件
- 换手机时用 ⬆ 导入备份文件恢复全部收藏

## 技术栈

纯前端 PWA：原生 JS (ES Modules) + CSS + IndexedDB + Service Worker。零依赖、零构建步骤。
